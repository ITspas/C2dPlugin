Cocos2d-x Plugins
-----------------------
### 命令工具使用
    1.安装C2dPlugin
        C2dTool plugin init platform
    2.卸载C2dPlugin
        C2dTool plugin uninit Platform
    3.安装插件
        C2dTool plugin add plugin platform
    4.删除插件
        C2dTool plugin rm plugin platform
